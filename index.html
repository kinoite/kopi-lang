Welcome to Kopi Language

Simple. Powerful. Expressive. Built for the modern developer.

Kopi is a new programming language designed with clarity and efficiency in mind. It aims to provide a straightforward syntax that is easy to learn for beginners, while offering the power and flexibility needed for complex projects.

Whether you're building web applications, command-line tools, or exploring new programming paradigms, Kopi provides a robust foundation.

Code Example


func main() {
    print("Hello, Kopi World!")

    let result = add(5, 3)
    print("The sum is: " + result)
}

func add(a: int, b: int) -> int {
    return a + b
}

struct Point {
    x: float,
    y: float
}

let p = Point{x: 10.5, y: 20.0}
print("Point coordinates: " + p.x + ", " + p.y)

Key Features

  • Clear and concise syntax, making code easy to read and write.
  • Strong, but flexible type system, catching errors early.
  • Designed for performance, with compilation to native code.
  • Modern concurrency features for parallel execution.
  • Cross-platform compatibility across various operating systems.
  • Rich standard library to accelerate development.
  • Active community and comprehensive documentation (on our Wiki!).
